# 概述

## 本项目基于开源软件 CNCJS 基础上进行二次开发, 专用于实现 MOOZ 的硬件控制

---

## 客户端下载地址

https://cn.dobot.cc/downloadcenter/dobot-mooz.html

## 将包括以下功能

- [ ] 激光雕刻

- [ ] CNC 雕刻

- [ ] 3D 打印

## 开发流程

`yarn` 安装依赖

`yarn start` 启动开发环境

`yarn ele:build` 打包 64 位 windows 客户端
